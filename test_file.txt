Тестовый документ для AI агентной системы

Этот документ содержит информацию о проекте FastAPI с React frontend.

Основные компоненты:
1. Backend на FastAPI с аутентификацией
2. Frontend на React с TypeScript
3. AI агентная система для анализа файлов
4. Система управления задачами

Технологии:
- Python 3.8+
- FastAPI
- SQLAlchemy
- React 18
- TypeScript
- Tailwind CSS

Функциональность:
- Регистрация и авторизация пользователей
- CRUD операции с задачами
- Загрузка и анализ файлов через AI агентов
- Чат с AI о содержимом файлов

Дата создания: 2024
Автор: Разработчик
Статус: В разработке

Важные заметки:
- Файлы изолированы между пользователями
- Поддерживаются форматы: txt, csv, json, md, html
- Максимальный размер файла: 10MB
- История чата сохраняется в памяти
