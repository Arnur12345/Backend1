Руководство по использованию FastAPI

FastAPI - это современный, быстрый веб-фреймворк для создания API с Python 3.7+ на основе стандартных подсказок типов Python.

Основные возможности:
1. Быстрая разработка: Увеличивает скорость разработки функций примерно на 200-300%
2. Меньше ошибок: Уменьшает количество ошибок, вызванных человеческим фактором, примерно на 40%
3. Интуитивно понятный: Отличная поддержка редактора с автодополнением везде
4. Простой: Разработан так, чтобы быть простым в использовании и изучении
5. Короткий: Минимизирует дублирование кода
6. Надежный: Получите готовый к продакшену код с автоматической интерактивной документацией

Установка:
pip install fastapi
pip install "uvicorn[standard]"

Простой пример:
```python
from fastapi import FastAPI

app = FastAPI()

@app.get("/")
def read_root():
    return {"Hello": "World"}

@app.get("/items/{item_id}")
def read_item(item_id: int, q: str = None):
    return {"item_id": item_id, "q": q}
```

Запуск сервера:
uvicorn main:app --reload

Автоматическая документация будет доступна по адресу:
- Swagger UI: http://127.0.0.1:8000/docs
- ReDoc: http://127.0.0.1:8000/redoc

Основные компоненты:
- Path Parameters: Параметры пути для динамических URL
- Query Parameters: Параметры запроса для фильтрации и пагинации
- Request Body: Тело запроса для POST/PUT операций
- Response Models: Модели ответов для валидации и документации
- Dependency Injection: Система зависимостей для переиспользования кода
- Security: Встроенная поддержка OAuth2, JWT и других схем безопасности

FastAPI автоматически генерирует OpenAPI схему и интерактивную документацию API.
